Vector an element of a Vector Space.


Inherits from: Object : Collection : SequenceableCollection : ArrayedCollection : RawArray : FloatArray


Represents an element of finite Vector Space of dimension N.  Usual operations are implemented.


Creation / Class Methods


*new (args)

Create new vector from a n-tuple of floats. Dimension of the vector is determined by the amount of floats.

Vector[1.2,0.3,5.6,3.2]


*rand (size, lo, hi)

random vector.

size - dimension of vector.

lo , hi - range of values

Vector.rand(4,-6.0,6.0);


*canonB (i, n)

ith element of canonic base of n-dimensional vector space.

Vector.canonB(0,6)


Accessing Instance and Class Variables

x,y,z

components for vectors up to 3 dimensions.

Vector[1,2,3].x

<|> (vector)

inner product.

Vector[1,2,3]<|>Vector[2,5,7]


// the canonical base is orthogonal

3.collect{ |i|  Vector.canonB(i, 3)<|>Vector.canonB((i+1).mod(3), 3) };


cross (vector)

cross product:

a × b = (a2b3 − a3b2) i + (a3b1 − a1b3) j + (a1b2 − a2b1) k = (a2b3 − a3b2, a3b1 − a1b3, a1b2 − a2b1).

note that the vectors should be 3 dimensional, and no error checking is done for speed.


Vector[1,2,3].cross(Vector[2,5,7])

// the cross product of the cannoic base elements folds into itself

3.collect{ |i|  Vector.canonB(i, 3) };

3.collect{ |i|  Vector.canonB(i, 3).cross(Vector.canonB((i+1).mod(3), 3)) };
